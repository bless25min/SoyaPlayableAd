# 先漲→反轉收下分析線（空單）訓練 Playable

這是一個 HTML5 Playable Ad 小遊戲，模擬先漲後反轉且收下分析線才進場空單的交易訓練流程。

## 遊戲玩法
1. 開啟 `index.html` 於瀏覽器。
2. 在開始畫面可以調整今日訊號數與播放速度。
3. 點擊 **開始模擬** 後，蠟燭圖會逐根播放。
4. 遇到訊號時需判斷是否符合「先漲→反轉且收下分析線」的計畫：
   - 符合：選擇進場空單。
   - 不符合：選擇放棄。
5. 若進場，系統會自動設定 SL/TP，並依走勢結算結果。
6. 所有訊號判斷完畢後會顯示成績單，可再玩一次。

## 可調參數
- **訊號數**：今日模擬產生的反轉訊號數量（3~5）。
- **播放速度**：K 棒自動播放的速度。

## 學習目標
- 制定交易計畫並嚴格遵守。
- 判斷 K 棒是否「收下/收上」分析線。
- 面對多個訊號時學會放棄不符合條件的情境。
- 透過 SL/TP 養成紀律化交易。

## 執行
不需額外安裝，下載專案後直接以瀏覽器開啟 `index.html` 即可。
